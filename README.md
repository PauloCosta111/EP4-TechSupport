
# EP4 - Scripts SQL para TechSupport Manager

## Como executar
1. Crie um banco de dados no MySQL ou PostgreSQL.
2. Execute o script `create_tables.sql` para criar as tabelas.
3. Execute o script `insert_data.sql` para popular as tabelas.
4. Execute o script `select_queries.sql` para testar consultas.
5. Execute o script `update_delete.sql` para atualizar e excluir dados.
